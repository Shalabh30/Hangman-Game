Pseudo Code Hangman Game

- Computer generates a random word and store it in a variable. The chosen word only includes lower case alphabets (no numbers, symbols, or space).

- Mask the coputer guess by dashes

- Display the length of word to user using dashes for each alphabet in the word.

- Prompt the user to guess a letter

- If the guess is correct, unmask the letter in computer guess

- If the guess is wrong, display incorrectGuess

- After each guess display the number of chances remaining

- If the incorrectGuess is equal to 12, prompt You Lost and exit the game.

- if the correctGuess is equal to Length of the computer guess, Prompt You Won and exit the game.
